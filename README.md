# Data Analytics Internship Projects

This repository contains the projects completed as part of the Data Analytics Internship Program. The projects focus on data cleaning, exploratory data analysis, customer segmentation, machine learning, and business insights generation using real-world datasets.

## 📌 Projects Included

### Task 1: Exploratory Data Analysis (EDA) on Retail Sales Data
#### Objective
Perform exploratory data analysis on retail sales data to uncover trends, patterns, and actionable business insights.

#### Key Activities
- Data Cleaning
- Descriptive Statistics
- Customer Analysis
- Product Analysis
- Sales Trend Analysis
- Data Visualization

#### Tools Used
- Microsoft Excel
- Python (Pandas, Matplotlib)
- Power BI
